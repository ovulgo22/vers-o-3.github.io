Glossário de Termos

Definições rápidas para os conceitos mais importantes em IA e Computação Quântica.

Algoritmo

Um conjunto de regras ou instruções passo a passo para resolver um problema ou realizar uma tarefa. A base de todo software.

Alinhamento (IA)

O desafio de garantir que os objetivos de um sistema de IA avançado estejam alinhados com os valores e intenções humanas.

Backpropagation

O algoritmo fundamental para treinar redes neurais. Ele calcula o gradiente da função de erro em relação aos pesos da rede.

Decoerência Quântica

A perda de informações de um sistema quântico para o ambiente. É o maior obstáculo para a construção de computadores quânticos estáveis.

Deep Learning

Um subcampo do Machine Learning baseado em redes neurais com muitas camadas (profundas), que permite aprender padrões complexos.

Emaranhamento Quântico

Um fenômeno no qual dois ou mais qubits se tornam interligados, de modo que o estado de um afeta instantaneamente o outro.

GAN (Rede Adversarial Generativa)

Uma arquitetura de IA com duas redes (um gerador e um discriminador) que competem entre si para criar dados sintéticos realistas.

LLM (Large Language Model)

Um tipo de modelo de IA treinado em vastas quantidades de texto para entender, gerar e interagir usando a linguagem humana. Ex: GPT-4.

Qubit (Quantum Bit)

A unidade básica de informação quântica. Pode existir nos estados 0, 1 ou em uma superposição de ambos, permitindo um poder de processamento massivo.

Rede Neural Artificial

Um modelo computacional inspirado na estrutura do cérebro humano, composto por nós (neurônios) interconectados em camadas.

Superposição

O princípio da mecânica quântica que permite a um qubit existir em múltiplos estados (0 e 1) ao mesmo tempo até ser medido.

Transformer

Uma arquitetura de rede neural baseada em mecanismos de "atenção", que se tornou a base para os LLMs mais avançados.
